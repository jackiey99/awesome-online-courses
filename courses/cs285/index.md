## 课程定位

这门课是当下 deep RL 最权威的研究生课程之一。它不是面向"应用工程师"的工具课，而是面向"做 RL 研究"的方法论课——会从问题定义开始，把 model-free、model-based、IRL、offline RL、meta RL 这些方向系统过一遍，并在每个方向给你足够的 paper-level 深度。

## 适合谁

- 已经有扎实的深度学习基础（PyTorch + 训练经验）
- 想做 RL 相关研究 / 工程
- 想理解 RLHF、AlphaGo、机器人学习这些当下热点背后的算法机制
